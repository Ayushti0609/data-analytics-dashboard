#  Data Analytics Dashboard

This is a simple and interactive data analytics dashboard built using Flask. The goal of this project is to make it easy to upload CSV files, explore data, and generate visualizations without any complex setup.

---

##  Features

- Upload CSV files easily
- View data preview in table format
- Get summary statistics instantly
- Generate interactive graphs (Bar, Line, Pie)
- Clean and responsive UI
- Smooth animations and loading experience

---

## Tech Stack

- Python (Flask)
- Pandas
- HTML, CSS, Bootstrap
- Chart.js (for interactive graphs)

---

## How It Works

1. Upload a CSV file  
2. The app processes the data using Pandas  
3. You can preview the data and summary  
4. Select a column and graph type  
5. Interactive chart is generated instantly
